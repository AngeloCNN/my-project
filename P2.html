泰坦尼克号数据的数据字典（中文版）解析：

数据来源：https://www.kaggle.com/c/titanic/data

PassengerId：游客编号

Survived：幸存情况

1：表示（幸存） 2：表示（遇难）

Pclass：船票类型

1：表示（头等舱） 2：表示（二等舱） 3：表示（三等舱）

Name：姓名

Sex：性别

Age：年龄

SibSp：与游客本人一同在船上的兄弟姐妹或者配偶

Parch：与游客本人一同在船上的父母或者孩子

Ticket：船票号码

Fare：船票价格

Cabin：船舱编号

Embarked：登船港口

关键变量解析：

Pclass:代表了社会经济地位。 1：代表（上层） 2：代表（中层） 3：代表（底层）

1.2：游客购买船票的等级与生还率的相关性验证¶

In [12]:
def correlation(x,y): #计算两组数据相关性的函数。                 
    std_x = (x - x.mean())/ x.std(ddof = 0)
    std_y = (y - y.mean())/ y.std(ddof = 0)
    return (std_x * std_y).mean()#皮尔逊积矩相关系数

Pclass = Pclass_Survived_mean.index.values#把生还率的索引值找出来然后转变成数组，也就是生还率对应的船票等级

print '船票类型对应生还率:',Survived,'\n'

1.2小结：通过相关性函数计算出，皮尔逊积矩相关系数约等于-0.99非常接近-1。通常情况下当系数接近1或者接近-1的情况下，我们会认为两组数据具有高度相关性。而本次计算结果非常接近-1，说明随着船票等级下降，游客的生还率也会随之大幅下降。虽然本次相关性的计算，每组数据的数据量只有3个，但是每组数据的统计计算都来自于船上的几乎所有乘客，因此此次相关性的验证计算是可靠而且有说服力的。

1推论一：由于数据显示船舱的等级与游客的生还率成正相关，因此可以推论，高级别的船舱可能拥有更多、更好、更容易拿到的救生设备，以至于在高等级船舱的游客更容易逃生。

1推论二：由于在数据来源的关键变量描述中，我们看到船票等级还是社会身份地位的象征，因此高等级船舱的游客生还率更高，不排除有在逃生和救援时由于身份地位不平等导致的救援顺序的差异因素。也就是说高等级船舱，除了有更好的救生设备，和更容易拿到的救生设备以外，逃生和救援的优先级别也许也是导致高等级船舱游客生还率高的原因。

9推论一：因为尊老爱幼是全世界人民的美德，因此当一个游客在船上携带多位老人或多个孩子的时候，目标会很醒目、很显眼，这样大部分船上的游客都可以看出他们是明显的弱势群体，因此携带多位老人或者孩子的乘客，更容易受到同船游客的帮助从而因为沾了老人和小孩的光而最先获救。

9推论二：当老人或者小孩的数量固定时，与游客随行的兄弟姐妹等青壮年越多，会逐渐在人群中失去弱势群体的身份，从而会失去更多获得额外帮助的机会和救生资源，使游客的生还率下降。

结论：¶

第一部分：决定船上游客生还率的最主要因素是船票的等级，船票的等级越高，游客的生还率越高。经过相关性验证，船票的等级与游客的生还率之间存在明显的正相关性。（经过相关性验证，结论具有可靠性）

第二部分：在相同等级的船舱中，游客普遍遵循女士优先，以及尊老爱幼的社会普世价值观，女性的生还率大于男性，孩子的生还率大于成年人，带老人的游客的生还率大于没有带老人的游客生还率。（此推论仅从数据可视化图表中直观看到，并未进行相关性验证，因此结论只是暂时的，有待进一步验证。）

第三部分：数据可视化显示有独立船舱的游客，生还率明显高于没有独立船舱的游客，原因可能是独立船舱内有更充足的救生设备，遇到危险时，没有人员的拥挤，更容易拿到救生设备，因此独立船舱的游客生还率更高。（1.此推论仅从数据可视化图表中直观看到，并未进行相关性验证，因此结论只是暂时的，有待进一步验证。2.此推论的假设前提是：['Cabin']列大部分数据缺失的部分本来就是没有独立船舱的，如果假设不成立，则此推论不成立，因此第三部分的推论仅供参考。）

第四部分：游客的生还率与游客登船的港口基本没有关系，最终还是船票等级决定了游客的生还率。S港口生还率最低是因为，S港口购买三等舱船票的游客数量最多。（此推论仅从数据可视化图表中直观看到，并未进行相关性验证，因此结论只是暂时的，有待进一步验证。）
